# 源网荷储开发工作台（Product Dashboard）

基于“页面是主对象、交付块标准化、角色视图组合展示”的工作台原型，已实现：
- 项目总览：项目介绍、用户流程、页面地图、产品范围、技术关注点、数据策略、质量验收与风险依赖
- 页面 Tab：`探索页`、`工作区页面`、`电站规划页`、`电站运行动画页`、`电站运行结果页`、`电站财务结果页`、`数据集页`
- 角色视图：全量 / 产品 / UIUX / 前端 / 后端 / 算法 / 测试 / 管理评审 / 自定义
- 页面内导航：摘要、关注重点、待开发页面模块 PRD、设计稿、接口、资源、折叠详情区
- 视图工具栏：支持按关键词搜索交付块、按已配置/缺失状态筛选
- 交付块统计：在页头显示当前角色视图的配置进度与缺失数量
- 设计稿挂载：支持图片预览和外链打开
- Mock 数据：外层展示可视化摘要，支持弹窗查看完整 Markdown
- 字段定义：按区域展示可视化表格，支持按字段或按分组复制 Markdown、下载 CSV 和 Excel
- 接口板块：支持 Swagger / Apifox / OpenAPI 链接与页面内嵌预览
- 缺失块提示：对未提供内容显示“未提供 / 待补充”
- 自定义视图：支持勾选显示块与上下调整顺序（保存在 `localStorage`）

## 启动方式

可直接用浏览器打开 `index.html`，也可在目录内启动本地静态服务：

```sh
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 目录说明
- `index.html`：页面骨架
- `styles.css`：样式与布局
- `app.js`：页面配置、角色视图配置、交互逻辑
